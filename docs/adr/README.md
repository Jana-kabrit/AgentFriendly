# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the `@agentfriendly` SDK. Each ADR documents a significant design decision, the context that led to it, the options considered, and the rationale for the choice made.

ADRs are numbered sequentially and never deleted — superseded ADRs are marked as such and a new ADR is created for the new decision.
